# 📚 Hermes Learning

Diário de aprendizado contínuo - documentação de descobertas e insights.

## 📁 Estrutura

```
hermes-learning/
├── daily/          # Aprendizados diários
├── weekly/         # Resumos semanais
├── topics/         # Tópicos organizados
│   ├── programming/
│   ├── ai-ml/
│   ├── devops/
│   └── tools/
└── insights/       # Insights valiosos
```

## 🎯 Objetivo

Manter um registro organizado de:
- Descobertas técnicas
- Soluções para problemas
- Melhores práticas
- Padrões e anti-padrões
- Recursos úteis

## 📝 Formato

Cada entrada inclui:
- **Data** - Quando foi aprendido
- **Contexto** - Situação que motivou
- **Aprendizado** - O que foi descoberto
- **Aplicação** - Como usar
- **Exemplos** - Casos práticos
